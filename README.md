## 🎨 ส่วนแสดงผล Computer Graphics

ตัวโปรเจกต์ประกอบไปด้วยตัวอย่าง **Interactive Computer Graphics** ที่เป็นการแปลงพิกัดรูปภาพสไปรต์สี 24-Bit (Transformation) และวาดลงบน HTML5 Canvas โดยควบคุมแบบ Real-time ด้วยคีย์บอร์ด

### 🎮 วิธีการควบคุม (Controls)

- **Translate (เลื่อนตำแหน่ง):** กดปุ่มลูกศร `↑` `↓` `←` `→`
- **Rotate (หมุนวัตถุ):** กดปุ่ม `Q` (ทวนเข็มนาฬิกา) หรือ `E` (ตามเข็มนาฬิกา)
- **Scale (ย่อ-ขยาย):** กดปุ่ม `Z` (ขยายใหญ่ขึ้น) หรือ `X` (ย่อเล็กลง)

## 🛠️ เทคโนโลยีที่ใช้

- **HTML5 & CSS3** สำหรับโครงสร้างเว็บสไตล์ Minimal Warm Tone (ใช้ Google Fonts "Outfit" และ "Prompt")
- **Vanilla JavaScript** สำหรับการประมวลผลและการคำนวณคณิตศาสตร์เมทริกซ์ 2D Transformation
- **HTML5 Canvas API** สำหรับแสดงผลแบบ Pixel-level
